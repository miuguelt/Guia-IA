# Proyecto Práctico Integral Final
## Módulo Bonus: IA para Presentaciones de Alto Impacto

---

## Visión General del Proyecto

### Objetivo
Los participantes conceptualizarán, generarán y perfeccionarán una presentación completa utilizando un flujo de trabajo con múltiples herramientas de IA, aplicando todos los conceptos aprendidos en los 5 pilares.

### Resultado Esperado
Una presentación profesional lista para usar, creada con al menos 3 herramientas de IA diferentes, que demuestre dominio de narrativa estratégica, diseño visual efectivo y preparación para entrega.

### Duración Estimada
- **Total**: 90-120 minutos
- **Fase 1 (Planificación)**: 15-20 min
- **Fase 2 (Creación con IA)**: 45-60 min  
- **Fase 3 (Refinamiento)**: 15-20 min
- **Fase 4 (Práctica y Evaluación)**: 15-20 min

---

## Estructura del Proyecto

### Fase 1: Planificación Estratégica (Pilares 1-2)

#### 1.1 Definición del Contexto
**Objetivo**: Establecer los parámetros fundamentales de la presentación.

**Actividades**:
1. **Selección de escenario** (elegir uno):
   - **Opción A (Laboral)**: Presentación de resultados trimestrales para dirección
   - **Opción B (Proyecto)**: Propuesta de nuevo proyecto/idea para aprobación
   - **Opción C (Personal)**: Presentación personal para entrevista o portfolio
   - **Opción D (Educativa)**: Capacitación sobre un tema de expertise

2. **Definición de parámetros clave**:
   - **Audiencia**: ¿Quién verá la presentación? (ejecutivos, equipo, clientes)
   - **Objetivo**: ¿Qué quiero lograr? (informar, persuadir, enseñar, motivar)
   - **Duración**: ¿Cuánto tiempo tendré? (5, 10, 15, 30 minutos)
   - **Mensaje central**: ¿Qué idea principal deben recordar?

3. **Selección de herramientas IA**:
   - Usar el **Comparador de Herramientas** para seleccionar:
     - 1 herramienta para generación de contenido/narrativa
     - 1 herramienta para diseño de slides
     - 1 herramienta para elementos visuales/gráficos
     - (Opcional) 1 herramienta para práctica/coaching

**Entregable Fase 1**: Documento de planificación con contexto definido y herramientas seleccionadas.

### Fase 2: Creación con IA (Pilar 3)

#### 2.1 Generación de Contenido y Estructura
**Objetivo**: Crear el esqueleto narrativo y contenido de la presentación.

**Actividades**:
1. **Uso del Generador de Prompts**:
   - Seleccionar template según tipo de presentación
   - Personalizar con parámetros de Fase 1
   - Generar prompt optimizado

2. **Creación de contenido**:
   - Copiar prompt a herramienta seleccionada (ChatGPT/Claude/Gemini)
   - Generar estructura detallada (títulos, puntos clave por slide)
   - Refinar iterativamente con prompts de ajuste

3. **Organización de narrativa**:
   - Estructurar en: Introducción, Desarrollo (3-5 puntos clave), Conclusión
   - Asegurar flujo lógico y persuasivo
   - Identificar transiciones entre secciones

#### 2.2 Diseño Visual con IA
**Objetivo**: Transformar el contenido en slides visualmente atractivas.

**Actividades**:
1. **Preparación para herramienta de diseño**:
   - Exportar contenido estructurado a formato compatible
   - Preparar brief de diseño (tono, estilo, colores)

2. **Generación de slides**:
   - Usar herramienta seleccionada (Gamma/Beautiful.AI/DeckRobot)
   - Importar contenido o usar prompt de diseño
   - Generar primera versión completa

3. **Creación de elementos visuales**:
   - Usar herramientas de diseño visual (DALL-E/Midjourney/Canva AI)
   - Generar imágenes para slides clave
   - Crear gráficos o iconos personalizados

#### 2.3 Integración y Primera Versión
**Objetivo**: Unir contenido y diseño en presentación coherente.

**Actividades**:
1. **Revisión de coherencia**:
   - Verificar que diseño apoye contenido
   - Asegurar consistencia visual
   - Chequear legibilidad y flujo

2. **Exportación**:
   - Guardar presentación en formato editable
   - Crear versión para compartir (PDF, enlace)

**Entregable Fase 2**: Presentación completa en primera versión, creada con al menos 3 herramientas de IA.

### Fase 3: Optimización y Personalización (Pilar 4)

#### 3.1 Evaluación Inicial
**Objetivo**: Identificar áreas de mejora en la presentación.

**Actividades**:
1. **Uso del Evaluador de Calidad**:
   - Cargar presentación (o describirla)
   - Obtener puntuación en 4 criterios
   - Revisar recomendaciones específicas

2. **Auto-evaluación**:
   - Revisar checklist de mejores prácticas
   - Identificar 3 fortalezas y 3 debilidades
   - Priorizar mejoras (alta/mediana/baja urgencia)

#### 3.2 Refinamiento con Técnicas Avanzadas
**Objetivo**: Aplicar técnicas de optimización aprendidas.

**Actividades**:
1. **Mejora de narrativa**:
   - Revisar y fortalecer mensaje central
   - Asegurar que cada slide tenga propósito claro
   - Mejorar transiciones y flujo

2. **Optimización visual**:
   - Aplicar principios de diseño (contraste, alineación, proximidad)
   - Mejorar legibilidad (tamaño de texto, contraste)
   - Asegurar coherencia de branding

3. **Personalización**:
   - Incorporar datos específicos o ejemplos relevantes
   - Adaptar tono para audiencia específica
   - Añadir elementos de engagement (preguntas, actividades)

#### 3.3 Iteración Final
**Objetivo**: Crear versión pulida lista para práctica.

**Actividades**:
1. **Aplicar mejoras**:
   - Implementar cambios priorizados
   - Verificar que todas las recomendaciones clave estén aplicadas

2. **Revisión final**:
   - Chequear ortografía y gramática
   - Verificar formato y compatibilidad
   - Asegurar que cumple con tiempo estimado

**Entregable Fase 3**: Presentación optimizada con reporte de mejoras aplicadas.

### Fase 4: Práctica y Evaluación Final (Pilar 5)

#### 4.1 Preparación para Entrega
**Objetivo**: Desarrollar habilidades de presentación efectiva.

**Actividades**:
1. **Creación de materiales complementarios**:
   - Generar handout/resumen con IA
   - Preparar notas para presentador
   - Crear Q&A anticipado

2. **Práctica con coaches de IA**:
   - Usar Simulador de Presentación para grabar práctica
   - Recibir feedback automático (tono, ritmo, claridad)
   - Practicar 2-3 veces incorporando feedback

#### 4.2 Evaluación Integral
**Objetivo**: Medir calidad final según criterios establecidos.

**Actividades**:
1. **Auto-evaluación final**:
   - Completar rúbrica de evaluación
   - Asignar puntuación en cada criterio
   - Justificar puntuaciones con ejemplos

2. **Evaluación por pares** (opcional):
   - Compartir presentación con compañero/mentor
   - Recibir feedback estructurado
   - Incorporar feedback final si aplica

3. **Reflexión final**:
   - Documentar aprendizajes clave
   - Identificar qué herramientas fueron más útiles
   - Planificar cómo aplicar lo aprendido en trabajo real

**Entregable Fase 4**: 
1. Grabación de práctica con feedback
2. Rúbrica de evaluación completada
3. Reflexión de aprendizajes

## Criterios de Evaluación del Proyecto

### Rúbrica de Evaluación (40 puntos total)

#### 1. Claridad del Mensaje (10 puntos)
- **10-9 pts**: Mensaje central extremadamente claro y memorable
- **8-7 pts**: Mensaje claro pero podría ser más impactante
- **6-5 pts**: Mensaje presente pero no completamente claro
- **4-0 pts**: Mensaje confuso o ausente

**Indicadores**:
- ¿El objetivo de la presentación es evidente desde el inicio?
- ¿Cada slide contribuye al mensaje central?
- ¿La conclusión refuerza claramente el mensaje?

#### 2. Innovación en el Uso de IA (10 puntos)
- **10-9 pts**: Uso creativo y estratégico de múltiples herramientas de IA
- **8-7 pts**: Uso efectivo de IA pero con oportunidades para mayor innovación
- **6-5 pts**: Uso básico de IA, principalmente como automatización
- **4-0 pts**: Uso mínimo o inefectivo de IA

**Indicadores**:
- ¿Se utilizaron al menos 3 herramientas de IA diferentes?
- ¿El uso de IA fue estratégico (potenciación) no solo automático?
- ¿Se integraron resultados de diferentes herramientas de manera coherente?

#### 3. Calidad del Diseño (10 puntos)
- **10-9 pts**: Diseño profesional, coherente y que mejora la comunicación
- **8-7 pts**: Diseño claro pero con oportunidades de mejora visual
- **6-5 pts**: Diseño funcional pero con problemas de legibilidad/coherencia
- **4-0 pts**: Diseño pobre que dificulta la comprensión

**Indicadores**:
- ¿Hay coherencia visual (colores, fuentes, layouts)?
- ¿El diseño apoya el contenido sin distraer?
- ¿Es visualmente atractivo y profesional?

#### 4. Efectividad Narrativa (10 puntos)
- **10-9 pts**: Narrativa persuasiva con flujo lógico y engagement alto
- **8-7 pts**: Buena estructura pero oportunidades para mayor impacto
- **6-5 pts**: Estructura básica presente pero flujo podría mejorar
- **4-0 pts**: Estructura confusa o narrativa poco efectiva

**Indicadores**:
- ¿La presentación tiene introducción, desarrollo y conclusión claras?
- ¿El flujo entre slides es lógico y natural?
- ¿La narrativa mantiene el interés y guía hacia una conclusión?

### Puntos Extra (hasta 5 puntos adicionales)
- **+2 pts**: Uso creativo de herramientas de IA no cubiertas en el módulo
- **+2 pts**: Integración efectiva de datos reales o personalización avanzada
- **+1 pt**: Creación de materiales complementarios de alta calidad

### Escala de Calificación Final
- **36-45 pts**: Excelente - Dominio completo de conceptos
- **30-35 pts**: Bueno - Aplicación efectiva con algunas áreas de mejora
- **24-29 pts**: Satisfactorio - Comprensión básica, aplicación limitada
- **0-23 pts**: En desarrollo - Necesita más práctica y comprensión

## Opciones de Proyecto por Contexto

### Opción A: Presentación Ejecutiva de Resultados
**Contexto**: Eres gerente de departamento presentando resultados trimestrales a la dirección.

**Requerimientos específicos**:
- 10-12 slides máximo
- Enfocada en datos y métricas
- Tono profesional pero accesible
- Incluir: contexto, logros, desafíos, próximos pasos

**Herramientas IA recomendadas**:
- ChatGPT/Claude para narrativa ejecutiva
- Gamma.app para diseño rápido y profesional
- Canva AI o DALL-E para gráficos de datos
- Beautiful.AI si se necesita diseño premium

### Opción B: Propuesta de Proyecto Innovador
**Contexto**: Presentas una nueva idea/proyecto para obtener aprobación y recursos.

**Requerimientos específicos**:
- 8-10 slides persuasivas
- Estructura: problema, solución, beneficios, implementación
- Tono convincente pero realista
- Incluir ROI estimado y timeline

**Herramientas IA recomendadas**:
- Claude para análisis profundo y ética
- Beautiful.AI para diseño persuasivo
- Midjourney para imágenes conceptuales
- Gamma.app para integración rápida

### Opción C: Presentación Personal para Entrevista
**Contexto**: Creas una presentación personal para entrevista de trabajo o portfolio.

**Requerimientos específicos**:
- 5-7 slides concisas
- Enfocada en valor único y logros
- Tono auténtico pero profesional
- Incluir: quién soy, logros clave, valor agregado
